Trabalho¶

Desenvolva os itens apresentados a seguir utilizando a linguagem de programação Python.

Item 1

Desenvola um algoritmo que leia três notas (\(N_1\), \(N_2\) e \(N_3\)) de um aluno, calcule e escreva a média final deste aluno. Considerar que a média (\(M\)) é ponderada e que o peso das notas é 2, 3 e 5. Fórmula para o cálculo da média final é: $\(M = \frac{N_1 \times 2 + N_2 \times 3 + N_3 \times 5}{10}\)$
(Fonte: CARVALHO, 2007)

Item 2

Peça ao usuário que informe o valor de um boleto, o percentual de juros cobrado e o número de dias em atraso. O programa deve calcular o valor total a ser pago, utilizando a fórmula $\(Total = Valor + Valor \times \frac{Juros}{100} \times Dias \)$
(Adaptado de LOPES, 2011)
