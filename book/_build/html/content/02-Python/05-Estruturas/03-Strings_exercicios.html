Exercício

A Cifra de César é uma das mais antigas e conhecidas técnicas de criptografia. Sua técnica consiste na substituição de cada letra do texto por outra, em que a letra é trocada pela letra seguinte no alfabeto com um número fixo. Por exemplo, para a frase python, e deslocamento utilizando \(3\) letras no alfabeto, o texto cifrado seria sbwkrq. Repare que: p->s,y->b,t->w,h->k,o->r e n->q.

Exercício

Implemente um algoritmo de criptografia que utiliza a Cifra de César, utilizando um deslocamento de \(3\) letras. Pergunte ao usuário qual frase deseja criptografar. Trabalhe apenas com letras minúsculas, mesmo que o usuário digite letras maiúsculas. Utilize a função ord() para obter valor ASCII de um determinado caractere, e a função chr() para obter o caractere a partir do seu valor ASCII.

Exercício
